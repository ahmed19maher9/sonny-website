About Sonny

Sonny is a powerful, privacy-focused personal AI assistant that runs entirely on your local machine. Built with C++ and powered by state-of-the-art language models, Sonny understands your voice, performs complex tasks, and helps you get things done — without sending your data to the cloud.

  • Fully Offline — All processing happens locally on your device. No cloud dependency, no data leaks.
  • Wake Word Detection — "Hey Sonny" activates the assistant, ready for your commands.
  • Multi-Tool Integration — From web search to file management, Sonny has a rich tool ecosystem.
  • Customizable — Extend Sonny with your own tools, voices, and behaviors.
  • Open Source — Check out the source code on GitHub and contribute to the project.
